DECOMPOSITION RULES:
1. Agents in the same parallel_group run concurrently
2. Higher priority = runs first (within sequential groups)
3. depends_on lists agents that must complete before this task starts

5. For L3 queries: 2-3 agents, some parallel
6. For L4 queries: 3+ agents, multi-step with dependencies
7. ALWAYS include risk_detector for policy analysis and recommendations
8. Claims + emotional distress: prioritize claims_guide, add regulation_engine for rights
