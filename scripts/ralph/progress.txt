## Project Overview
- Backend: FastAPI (Python 3.12) in botproject/
- Database: MongoDB (Atlas) + Redis
- AI: OpenAI, LangChain, LangGraph
- Frontend: Flutter (in frontend/)

## Key Patterns Discovered

### Code Structure
- Routers are in botproject/routers/
- Services are in botproject/services/
- Models are in botproject/models/
- Core utilities in botproject/core/

### Important Files
- app.py: Main FastAPI application, router registration
- core/config.py: Settings and environment config
- core/middleware.py: CORS, GZIP, security middleware
- core/dependencies.py: Dependency injection functions
- core/rate_limiter.py: Redis-backed rate limiting
- services/token_service.py: JWT token management (NEW)
- routers/token.py: Token refresh endpoints (NEW)

### Conventions
- Routers use @limiter.limit(RATE_LIMITS["key"]) for rate limiting
- Services are singleton instances
- Models use Pydantic with Field() for descriptions
- HTTPException with dict detail for errors
- Logging via Python logging module

### Testing
- No pytest configured currently
- Use python3 -m py_compile to check syntax
- Verify imports with: python3 -c "from app import app"
